Google Sheet Auto-Score Setup

🤖 Auto-Updating Scores

Once configured, this sweepstake updates itself — even while you're on holiday! Scores are fetched from football-data.org (free API) into a Google Sheet, and the pages read from it every 5 minutes.

  1. Register for a free API key at football-data.org (takes 2 minutes)
  2. Create a new Google Sheet, go to Extensions → Apps Script
  3. Paste the entire contents of wc2026-sheet-script.js (included in the download)
  4. Edit the API_KEY constant at the top with your key
  5. Run setupSheet first, then createHourlyTrigger
  6. Publish the sheet: File → Share → Publish to web → Matches sheet → CSV
  7. Paste the published URL below and click Save

Current: https://script.google.com/macros/s/AKfycbwHEvMX5jYIZqFHuBoqg8NqYdT7j1i3AYX_McQjNZ93XbUzuGu5SN5UXGnG90HVbYSV/exec

ℹ️ How the auto-update works

⏰  Google Apps Script runs every hour in the cloud — no PC needs to stay on
📡  It calls football-data.org, gets all 104 match results as they happen
📊  Results are written to your Google Sheet automatically
🌐  Anyone opening the bracket page fetches the sheet and shows live scores
✈️  Works completely unattended — perfect for when you're on holiday!
🔧  You can still manually override any result using the Results tabs below
